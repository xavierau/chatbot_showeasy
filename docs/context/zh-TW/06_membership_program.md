## 商業規則與細節

### 優惠不疊加
會員折扣通常唔可以同其他推廣代碼一齊用。

### DDC 限定
8折/9折 購票優惠主要適用於 **DDC 原創節目**。

### 合作夥伴
唔同餐廳、按摩店嘅優惠可能有唔同，記得睇返 App 入面嘅詳情頁面。

### 會員續費
- 會員資格為期一年
- 到期前會收到續會提醒
- 金會員續會有額外現金券獎勵

### 退款政策
- 會員費一經繳交不設退款
- 特殊情況可聯絡客服處理
